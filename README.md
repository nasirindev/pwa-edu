# Media Belajar SD - Interactive Learning PWA

A modern, professional, and kid-friendly Progressive Web Application (PWA) designed for elementary school students. This platform provides interactive learning modules and quizzes to make education an adventurous journey.

## 🚀 Tech Stack

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Database**: [SQLite](https://sqlite.org/)
- **ORM**: [Prisma](https://www.prisma.io/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **PWA**: Native Service Worker integration with Next.js Metadata API

## ✨ Key Features

- **Installable (PWA)**: Can be installed on Android, iOS, and Desktop for a native app-like experience.
- **Offline Support**: Core assets and lessons are cached for access without an internet connection.
- **Kid-Friendly UI**: High-contrast colors, rounded typography (Quicksand), and playful animations.
- **Interactive Quizzes**: Real-time feedback with confetti effects and score tracking.
- **Fast Performance**: Optimized using Next.js Server Components and Stale-While-Revalidate caching.
